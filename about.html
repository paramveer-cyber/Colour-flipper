This is a simple Colour-Flipper you can press the Background Color button on any page and it Will flip the background Colour.

A Colour-Flipper is a code which randomly choses any colour between ["Red", "Green", "Blue", "Yellow", "Purple", "Lightblue"] and sets it as the background of the Page.

Hex color codes are values that tell the display how much of a color to show. Unlike the normal Colour-Flipper it is not limited to any set amount of colours.
